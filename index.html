Transferencias Internacionales

Realice la revision y modificacion de una parte del codigo en un sistema encargado de calcular las comisiones e IVA de las transacciones Internacionales en la empresa financiera. (Por razones de contrato no se me permite dar mas datos :)

Sistemas de camara de vision

El sistema se constituye de una aplicación desarrollada completamente en Python. Su objetivo primordial es emplear una cámara para capturar una imagen y posteriormente compararla con otra imagen previamente adquirida. Este procedimiento implica una comparación píxel a píxel, generando como resultado un porcentaje de similitud. La aplicación incluye una interfaz gráfica de usuario que proporciona funcionalidades tales como inicio de sesión, múltiples puntos de control y la capacidad de seleccionar el color de la pieza para llevar a cabo la comparación.
